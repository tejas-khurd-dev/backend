# ⚙️ Backend Workbench

A structured collection of projects, notes, and experiments built while learning backend development step by step through hands-on implementation.

---

## Philosophy

This repository is not about watching tutorials.

The focus is to **understand backend systems by building, breaking, and refining them**.
Each concept is implemented practically to develop real intuition about how servers, APIs, and databases work.

---

## Learning Focus

* Server architecture
* REST API design
* Routing and middleware
* Authentication & authorization
* Database design 
* CRUD operations
* Error handling
* Security basics
* Performance optimization

### 🔗 REST API

## Getting Started

Clone the repository:

```bash
git clone https://github.com/tejas-khurd-dev/backend
```

Navigate into the repository:

```bash
cd backend-workbench
```

Go into any module:

```bash
cd "1 create-server"
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
